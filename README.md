# Weather-Station-RP

## Ambiente
Este sitio es una página estática basada en la plantilla Clean Blog de Start Bootstrap. Solo utiliza HTML, CSS y JavaScript del directorio del proyecto junto con fuentes y scripts cargados desde CDNs; no hay dependencias ni pasos de compilación adicionales.

## Cómo probarlo localmente
1. Abre `index.html` directamente en tu navegador, o
2. Sirve el directorio con un servidor estático, por ejemplo:
   ```bash
   python3 -m http.server 8000
   ```
   Luego visita http://localhost:8000 en tu navegador.
